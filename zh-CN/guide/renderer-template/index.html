RendererTemplate 扩展表单项 ​

通过 FormItems 表单项 , 我们知晓 templateType 决定渲染哪个组件。

接下来，我们将学习如何自定义 templateType

FormItemRendererTemplate ​

html
<VkfForm>
  <template #rendererTemplate>
    <VkfFormItemRendererTemplate type="YOUR_TEMPLATE_TYPE">
      <...>
    </VkfFormItemRendererTemplate>
  </template>
</VkfForm>

参考以上代码, 可以通过 VkfFormItemRendererTemplate 组件自定义 templateType.

你完全可以在你的项目中定义一个新的 Vue 组件，来扩展项目所需的表单项。

本质上这一设计遵循策略模式，即通过 templateType 匹配 VkfFormItemRendererTemplate 的 type 属性，来渲染插槽中的内容。

FormItemRendererTemplate Slots ​

在定义 templateType 时，使用 formItem 中的数据是关键。

FormItemRendererTemplate 默认插槽中, 提供了需要的数据, 以完成表单项配置和组件的通信。

完整的插槽参数
ts
export interface DefaultSlotArguments {

  raw: RendererTreeData

  /**
   * 根据 templateIf 生成的控制显隐事件
   */
  templateIf: (e?: NormalObject) => boolean 

  /**
   * 当前表单项的数据，排除渲染占用字段
   */
  props: Omit<RendererTreeData, 'templateType' | 'templateIf'>

  /**
   * 当前表单项，在整个 formItems 中的位置
   */
   k: (string|number)[]


  /**
   * FormItemRenderer 便于子组件使用
   */
  Renderer: typeof VkfFormItemRenderer

  
  /**
   * 对应的表单数据（如果formItem提供prop)
   */
  value?: never


  /**
   * 更新表单数据（如果formItem提供prop)
   */
  input?: (v: any) => void

  /**
   * 从表单数据中获取对应的值
   */
  getDataValue: (key: MaybeArray<number | string>) => any

  /**
   * 更新表单数据
   */
  emitSetData: (e: SetDataEvent) => void

}

另外，@vunk/form 中还提供了一些 备用表单项 , 他们虽不是默认表单项。

但你可以通过 VkfFormItemRendererTemplate 来扩展表单项，以便你更好的构建项目。

扩展规范 ​

Form 组件是基于 ElementPlus Form 组件的封装的。

我们在扩展表单项时，需要遵循以下规范：

  • 使用 FormItem 组件包裹表单输入控件（含 prop）, 保证 ElementPlus 的原生特性，如表单验证等。

    vue
    <VkfFormItem 
  v-bind="formItemBindProps"
>   
  <ElInput>
    <!-- ... -->
  </ElInput>
</VkfFormItem>

  • 暴露 Source 类型，以便扩展 FormItem 类型

    ts
    import { BasicSource } from '@vunk/form'
import type { VueComponentPropsType } from '@vunk/core'
export interface Source<P extends string = string> 
extends VueComponentPropsType<typeof VkfInput>, BasicSource { 
  prop?: P | P[],
  templateType: 'VkfInput',
}

  • 使用 readonly 字段，使得 Form 控制所有表单项的只读状态。

    ts
    import { useComputedReadonly } from '@vunk/form/composables'
export default {
  props: {
    readonly:{
      type: Boolean,
      // 重置为 undefined 时，以便未定义时，使用 Form 的 readonly
      default: undefined,
    }
  },
  setup(props) {
    const readonly = useComputedReadonly(props)
    return { readonly }
  }
}
完整的 Input 表单项组件, 用于参考
vue
<script lang="ts">
import { props, emits } from './ctx'
import { defineComponent } from 'vue'
import { ElInput } from 'element-plus'
import { VkfFormItem, _VkfFormItemCtx } from '@vunk-form/components/form-item'
import { createInputBindProps, createInputOnEmits } from '@vunk-form/shared/element-plus'
import { useComputedReadonly } from '@vunk-form/composables'

export default defineComponent({
  name: 'VkfInput',
  components: {
    ElInput, VkfFormItem,
  },
  props,
  emits,
  setup (props, { emit }) {
    const inputBindProps = createInputBindProps(props, ['readonly'])
    const inputOnEmits = createInputOnEmits(emit)
    
    const formItemBindProps = _VkfFormItemCtx.createBindProps(props)
    const readonly = useComputedReadonly(props)

    return {
      inputBindProps,
      formItemBindProps,
      readonly,
      inputOnEmits,
    }
  },
})
</script>

<template>
  <VkfFormItem 
    v-bind="formItemBindProps"
  >   
    <ElInput
      v-bind="inputBindProps"
      :readonly="readonly"
      v-on="inputOnEmits"
    >
      <template 
        v-for="(item, key) of inputSlots" 
        :key="key"
        #[key]="e"
      >
        <component :is="item?.(e)"></component>
      </template>

      <slot></slot>
    </ElInput>
    <slot name="formItem"></slot>
  </VkfFormItem>
</template>
ts
// ctx.ts
import { inputProps, inputEmits } from 'element-plus'
import { _VkfFormItemCtx } from '@vunk/form/components/form-item'
import { PropType } from 'vue'
import { InputSlots } from './types'
import type {} from 'element-plus/es/utils'
import { bindPropsFactory, onEmitsFactory } from '@vunk/core/shared/utils-vue'

export const props = {
  ...inputProps,
  ..._VkfFormItemCtx.props,
  maxlength: {
    type: Number,
    required: false,
  },
  minlength: {
    type: Number,
    required: false,
  },
  inputSlots: {
    type: Object as PropType<InputSlots>,
    default: undefined,
  },
  rows: {
    type: Number,
    default: undefined,
  },
}
export const createBindProps = bindPropsFactory(props)


export const emits = {
  ...inputEmits,
}
export const createOnEmits = onEmitsFactory(emits)
ts
// types.ts
import type { Func, VueComponentPropsType } from '@vunk/core'
import { BasicSource } from '@vunk/form'
import VkfInput from './index.vue'
export type InputSlots = {
  prefix?: Func
  suffix?: Func
  prepend?: Func
  append?: Func
}

export interface Source<P extends string = string> 
extends VueComponentPropsType<typeof VkfInput>, BasicSource { 
  prop?: P | P[],
  templateType: 'VkfInput',
}
vue
<!-- 扩展 -->
<VkfFormItemRendererTemplate type="VkfInput">
  <template #default="{ props, value, input }">
    <VkfInput 
      :model-value="value" 
      v-bind="props" 
      @update:model-value="input"
    ></VkfInput>
  </template>
</VkfFormItemRendererTemplate>
